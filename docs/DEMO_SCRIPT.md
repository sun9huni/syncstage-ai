# SyncStage AI - 피칭 대본 (3분 데모)

> **🎤 발표자 준비사항:** 
> - 로컬 환경에 `npm run dev` 구동 확인 (`localhost:3000`)
> - 화면 우측 패널의 채팅 입력창(Patch) 활성화 확인
> - 볼륨 체크 완료

---

## ⏱️ 0:00 - 인트로 (Problem & Solution)
"안녕하세요, K-pop 무대 기획의 혁신, **SyncStage AI** 팀입니다.
완벽한 아이돌 무대를 만들기 위해 A&R 디렉터와 안무가, 비주얼 팀이 매번 수십 번씩 피드백을 주고받는 과정을 아시나요? 
음원 하이라이트 구간에 어떤 안무가 들어가야 할지, 어떤 의상이 어울릴지 소통하는 비용은 막대합니다.

저희는 **Gemini 2.0 Flash와 Function Calling 기술을 활용해, 음원을 넣기만 하면 즉각적으로 초안을 짜고 자연어로 수정할 수 있는 AI A&R 디렉터**를 만들었습니다."

## ⏱️ 0:30 - 데모 1: 음원 업로드 & 초안 생성 (The Magic)
"(화면을 띄우며) 자, 여기 새로운 데모 음원이 도착했습니다. 
보시다시피 별도의 수동 작업 없이 음원을 업로드하면, Gemini가 오디오의 BPM과 분위기를 분석하여 **[초안 스토리보드]**를 짜냅니다.

(재생 버튼 클릭)
음악이 나오면서 좌측의 3D 아바타를 보세요. 
분석된 인텐시티(Intensity)와 모션(ClipId)에 맞춰, 5초 구간의 잔잔한 바운스에서 15초 하이라이트 구간의 강렬한 팝핀으로 **자동으로 안무가 전환(Sync)**됩니다."

## ⏱️ 1:20 - 데모 2: 자연어 패치 & 리비전 (Function Calling)
"기존 생성형 AI의 한계는 **'수정의 어려움'**이었습니다. 한 부분만 고치려다 전체가 망가지곤 하죠.
저희의 핵심 경쟁력은 **'의도(Intent) 기반 패치'**입니다.

(채팅창에 입력: '도입부 안무를 약간 힙합 그루브로 바꾸고, 분위기를 더 어둡게 해줘')
A&R 디렉터처럼 피드백을 주면, Gemini는 전체 데이터를 새로 쓰는 것이 아니라 **Function Calling**을 통해 `update_segment`와 `update_style` 도구를 정확히 골라내어 **핵심 구간만 타겟하여 수정**합니다. 

(화면 우측 상단 지목)
보이시나요? 우측 상단의 **Revision(버전)**이 방금 '1'로 올라가며 데이터 무결성을 증명합니다."

## ⏱️ 2:10 - 데모 3: 비주얼 컨셉 시각화 (Nano Banana 연동 비전)
"안무가 정해졌으니, 무대 의상을 볼까요?
(우측 하단의 [Visualize Concept] 버튼 클릭)
Gemini가 도출한 'Y2K Streetwear'라는 프롬프트를 바탕으로, 무대 비주얼 컨셉 시안이 즉각 렌더링 됩니다. 
이 과정에서 생성된 상세 프롬프트는 추후 **Nano Banana와 같은 전문 이미지 제너레이터에 바로 넘길 수 있는 고품질 데이터**로써 기능합니다."

## ⏱️ 2:40 - 아웃트로 (Vision)
"SyncStage AI는 단순한 안무 생성기가 아닙니다. 
**기획자, 안무가, 스타일리스트가 하나의 시각화된 보드를 보며 자연어로 소통하게 만드는 차세대 협업 툴(Collaboration Tool)**입니다.

오직 Gemini 2.0의 추론 능력과 도구 제어 능력이 있었기에 가능한 속도와 안정성이었습니다.
음악을 눈으로 보게 만드는 싱크스테이지였습니다. 감사합니다!"
